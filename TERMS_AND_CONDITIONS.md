## 3. Description of the Bot

The Bot is a multi-purpose Discord application that provides the following features:

### 3.1 Economy and Progression
- A coin and gem currency system with wallets, daily rewards, and streaks.
- A levelling system based on server activity, with rank cards and tiered ranks.
- Achievements and quest systems with milestone rewards.
- A bank system with savings accounts, deposits, withdrawals, tiered interest, peer-to-peer transfers, and loans.
- An economy graph and spending analytics.

### 3.2 Mini-Games
- Gambling games: coinflip, slots, roulette, blackjack, hilo, five-dice.
- Skill games: Wordle, Hangman, Trivia, Riddles, Country Quiz, Emoji Phrase, Unscramble, Mastermind, Memory/Pairs, Math Sprint, Solo Heist, Hide & Seek, Number Hunt, Lottery.
- A rock-paper-scissors game and a dice roller.

### 3.3 AI Systems
- An AI chat system supporting multiple providers (Groq, Cerebras, OpenRouter) and ten personality modes. The AI responds to messages in configured channels.
- An Akinator-style guessing game powered by AI and a Wikipedia-crawled knowledge base.
- AI-generated trivia questions, riddles, and country quizzes.

### 3.4 Voice Features
- Speech-to-text (STT) transcription via Groq's Whisper API while in voice channels.
- Text-to-speech (TTS) playback using the Murf API.
- Opt-in voice cloning, which records a sample of your voice and uses it for TTS output.
- A soundboard system for playing preset audio clips in voice channels.

### 3.5 Server Management and Social
- Customisable welcome and leave messages with canvas-rendered welcome cards, DM messaging, invite tracking, and auto-role assignment.
- Reaction roles with configurable mappings.
- Leaderboards for coins, XP, bank balance, achievements, and game stats.
- A server pet with stats, moods, and interactive events.
- A time capsule system for scheduling messages to be delivered in the future.
- Auto-quote channels.
- GIF reactions (fun and anime categories).
- Image and media processing tools (filters, captions, overlays, transformations).
- A guidebook and per-server setup/configuration panel.
- Custom embed builder and custom command manager.
- Event scheduling and announcement system.

### 3.6 Customisation
- Per-user card personalisation: rank card theme, style, accent colour, and background tint.
- Per-guild prefix, economy parameters, and embed colour branding.

### 8.2 Voice Cloning Opt-In

Voice cloning is an opt-in feature. By using the voice clone opt-in command, you explicitly consent to:
- A sample of your voice being recorded.
- That sample being transmitted to and processed by the Murf API.

You may withdraw your consent and request deletion of your voice clone at any time using the removal command.
